## How To Read This

- **Route accuracy** compares each decision to the fixture's labeled expectation.
- **Unsafe reuse** counts direct cache returns that fail continuity gates. It is a
  tripwire and should be zero while routing behaves.
- **Drift** accumulates weighted penalties for changed continuity dimensions; the
  reuse gate is drift <= 0.25.
- **Cost** figures are normalized planning estimates, not vendor pricing. Metric
  definitions and blind spots are documented in `docs/eval_card.md`.
